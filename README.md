# XMind2Pdf

## Install
```shell
$ git clone <this repository>
$ npm link
```

## Usage
```shell
$ xmind2pdf
➜ xmind2md 
? Which Xmind File Convert Markdown? (Use arrow keys)
❯ filename1.xmind 
  filename2.xmind 
? Where is Convert File Destination? (./tmp/output.md)
```

## Caution
I dropped down the XMind library version to use methods of reading XMind files.
